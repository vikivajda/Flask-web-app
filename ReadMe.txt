-Python letöltése: https://www.python.org/downloads/
-Letöléskor "Add Python to PATH" kipipálása
-Parancssorba begépeljük a "python --version", majd a "python get-pip.py" kifejezést
-VSC vagy tetszőleges környezet letöltése
-Környezeten belül terminálba "pip install flask", majd "pip install flask-login" és "pip install flask-sqlalchemy"
